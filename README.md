## 🏗️ Featured Projects

### 🤖 [Shehab Store Assistant](./Shehab)
*AI Store & E-commerce Agent* | An LLM-powered agent developed with **Flowise** to provide intelligent store assistance, product searches, and metadata analysis.
- **Tech**: Flowise, LLM Agents, Node.js (Custom Tools)
- **Key Feature**: Custom API tools for real-time product/package retrieval and collection filtering.

### 🚦 [Traffic Management System](./Traffic%20Management%20System)
*Graduation Project* | A high-performance traffic signal controller using **YOLOv11** for real-time car density analysis.
- **Tech**: YOLOv11, OpenCV, Gradio
- **Key Feature**: Dynamic timing based on vehicle counts and license plate OCR.

### 🤖 [Discord AI Bot](./Discord%20AI%20bot)
A comprehensive Discord interface for **Gemini 2.0 Flash**, featuring OCR and PDF processing.
- **Tech**: Discord.py, Gemini 2.0, Tesseract OCR
- **Key Feature**: AI-driven file reading and intelligent chat context.

### 🖼️ [Image Captioning](./Image%20caption)
Extremely accurate image description generator based on the **BLIP-2** architecture.
- **Tech**: Salesforce BLIP-2, PyTorch, Transformers
- **Key Feature**: High-quality caption generation with Beam Search.

### 🌐 [Web AI Chat](./Web%20AI%20Chat)
Interactive web-based chatbot using the **BlenderBot** model.
- **Tech**: Flask, Transformers, REST API
- **Key Feature**: Full-stack integration for transformer models.
